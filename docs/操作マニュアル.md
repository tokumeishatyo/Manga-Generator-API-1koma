## はじめに

### このマニュアルについて

このマニュアルは、**絵を描く技術がなくても、自分のイメージを画像にしたい**という創作者のためのガイドです。

AI創作工房のAIは、「**優秀だが、指示待ちの新人アシスタント**」と考えてください。明確に指示すればするほど、期待通りの結果が返ってきます。逆に、曖昧な指示では思わぬ結果になることもあります。

### 基本的な考え方

**「ガチャ」ではなく「ディレクション」**

1回で完璧な画像が出ることは稀です。**生成→選択→修正（精錬）**というプロセスを繰り返して、理想の画像に近づけていきましょう。

## 1. 起動と初期設定

### 1.1 アプリの起動

```
【図を挿入: 起動画面のスクリーンショット】
```

1. `main.py`を実行してアプリを起動します
2. 初回起動時にAPIキー入力画面が表示されます

### 1.2 APIキーの設定

Google AI APIキーが必要です。

**取得手順:**
1. [Google AI Studio](https://aistudio.google.com/apikey)にアクセス
2. Googleアカウントでログイン
3. 「Get API Key」でキーを取得
4. アプリの設定画面にキーを入力

**注意:** APIは従量課金制です。無料枠内での使用を推奨します。

### 1.3 出力タイプの違い

| タイプ | 説明 | 使い分け |
|--------|------|----------|
| **YAML出力** | YAMLをクリップボードにコピー | ブラウザ版Geminiで使いたい時 |
| **通常モード** | アプリ内で直接画像生成 | 標準的な使い方 |
| **清書モード** | 参考画像+YAMLで高品質再描画 | 気に入った画像の高解像度化 |
| **シンプルモード** | テキストだけで生成 | 手軽に試したい時 |

**YAML出力と直接生成の使い分け:**
- **直接生成（通常/清書/シンプル）**: アプリ内で完結、手軽
- **YAML出力**: ブラウザ版Geminiにコピペして使用。ブラウザ版で良い結果が出たら、その画像を清書モードで高解像度化できます

### 1.4 最初の生成テスト

設定が完了したら、シンプルモードで動作確認しましょう。

1. 出力タイプで「シンプルモード」を選択
2. テキスト欄に「青い空と草原」と入力
3. 「生成」ボタンをクリック
4. 10〜30秒待つと画像が表示されます

```
【図を挿入: シンプルモードでの初回テスト画面】
```

## 2. 画面の見方

### 2.1 メイン画面の構成

```
【図を挿入: メイン画面の各部説明付きスクリーンショット】
```

| エリア | 役割 |
|--------|------|
| **左列** | 出力タイプ選択、基本設定、機能ボタン |
| **中央列** | YAMLプレビュー（AIへの指示書）、参照画像プレビュー |
| **右列** | 生成画像プレビュー、保存・加工ボタン |

### 2.2 出力タイプの選び方

| タイプ | いつ使う？ |
|--------|-----------|
| **シンプルモード** | 手軽に1枚絵を作りたい時 |
| **通常モード** | キャラクターや背景を段階的に作る時 |
| **清書モード** | 気に入った画像を高解像度で再生成したい時 |
| **YAML出力** | ブラウザ版Geminiで使いたい時 |

### 2.3 共通設定

すべての生成で設定できる項目：

**アスペクト比（画像の縦横比）**
| 設定 | 向き | 用途例 |
|------|------|--------|
| 1:1 | 正方形 | SNSアイコン、汎用 |
| 16:9 | 横長 | サムネイル、プレゼン |
| 9:16 | 縦長 | スマホ壁紙、4コマ漫画 |
| 3:1 | 超横長 | 三面図専用 |

**解像度**
- **2K**: 高解像度（印刷用、完成品向け）
- **1K**: 標準（下書き、確認用）

**カラーモード**
- フルカラー、モノクロ、セピア、デュオトーン

## 3. まずは主人公を作ってみよう

キャラクターを作る際は、**顔→素体→衣装→ポーズ**の順で段階的に作成します。この流れで作ることで、**一貫性のあるキャラクター**が出来上がります。

### 3.1 Step1: 顔を作る

キャラクターの顔立ちを決めます。

```
【図を挿入: 顔三面図設定画面】
```

**操作手順:**
1. 「顔三面図」ボタンをクリック
2. 設定ウィンドウでキャラクター情報を入力：

| 項目 | 入力例 | ポイント |
|------|--------|----------|
| キャラクター名 | 彩瀬こよみ | 管理用の名前 |
| 外見説明 | ショートボブの茶髪、大きな青い瞳、元気な笑顔 | 特徴を箇条書きで |
| 描画スタイル | 標準アニメ風 | 作品の雰囲気に合わせて |

3. 「YAML生成」ボタンをクリック
4. メイン画面で「生成」をクリック

**コツ:**
- 髪飾りやホクロなど、**識別しやすい特徴**を入れておくと、後のステップでもキャラが維持されやすくなります
- 気に入った顔ができたら、すぐに画像を保存しましょう

### 3.2 Step2: 体型を決める

顔を固定したまま、体型を作ります。

```
【図を挿入: 素体三面図設定画面】
```

**操作手順:**
1. 「素体三面図」ボタンをクリック
2. 設定ウィンドウで：
   - 「顔三面図を指定」でStep1の画像を選択（**必須**）
   - 体型プリセットを選択
3. 「YAML生成」→「生成」

**重要:** 顔三面図を必ず指定してください。指定しないと別人になってしまいます。

### 3.3 Step3: 衣装を着せる

2つの方法があります：

#### 方法A: プリセットから選ぶ（おすすめ）

```
【図を挿入: 衣装ビルダー画面】
```

1. 「衣装着用三面図」ボタンをクリック
2. 「服装ビルダーを使用」をオンに
3. カテゴリを選択（学生服、カジュアル、ファンタジー等）
4. 色・柄・スタイルを指定
5. 素体三面図のパスを指定

#### 方法B: 参考画像から着せる

別の画像の衣装をキャラに着せたい場合：

1. 「参考画像から衣装をコピー」をオンに
2. 着せたい衣装の画像を指定
3. フィットモードを選択：
   - **素体優先**: キャラの体型を維持
   - **衣装優先**: 衣装のシルエットを維持
   - **ハイブリッド**: バランス重視

```
【図を挿入: フィットモード比較図】
```

### 3.4 Step4: ポーズをつける

動きのあるキャラクターを作ります。

```
【図を挿入: ポーズ設定画面】
```

**操作手順:**
1. 「ポーズ三面図」ボタンをクリック
2. 衣装三面図のパスを指定（**必須**）
3. ポーズプリセットを選択 or カスタム入力
4. 表情・視線・風エフェクトを設定
5. シーン合成に使う場合は「背景透過」をオンに

**ポーズプリセット例:**
- 波動拳（かめはめ波）
- スペシウム光線
- ライダーキック
- 坐禅（瞑想）

**ポーズキャプチャ（下書きからポーズを取り込む）:**
写真やラフスケッチからポーズだけを取り込むことができます：
1. 「ポーズキャプチャ」をオンに
2. 参考にしたいポーズの画像を指定（写真、イラスト、棒人間でもOK）
3. キャラの外見は維持したまま、ポーズだけが適用されます

### 3.5 角度を変える

ポーズ三面図から、特定の角度の1枚絵を抽出します。

```
【図を挿入: 角度変更画面】
```

| 角度 | 効果 |
|------|------|
| 正面 | そのまま |
| 左斜め前 / 右斜め前 | 斜め45度（立ち絵向き） |
| 上から見下ろす | 俯瞰（威圧感を出す） |
| 下から見上げる | 煽り（迫力を出す） |

| ズーム | 用途 |
|--------|------|
| 全身 | 立ち絵、アクションシーン |
| 上半身 | 会話シーン |
| バストアップ | キャラ紹介 |
| 顔アップ | 感情表現 |

## 4. 世界観を作ろう

### 4.1 背景を作る

キャラクターの舞台となる背景を作成します。

```
【図を挿入: 背景生成設定画面】
```

**基本的な使い方:**
1. 「背景生成」ボタンをクリック
2. 背景説明を入力、またはプリセットを選択
3. 「YAML生成」→「生成」

**背景説明のコツ（ト書きを書く感覚で）:**

| 良い例 | 悪い例 |
|--------|--------|
| 夕暮れの屋上、オレンジ色の空、手すり、遠くにビル群 | 屋上 |
| 雨の夜のコンビニ前、濡れたアスファルト、店内の光が漏れる | コンビニ |

**プリセット一覧:**
- 学校（教室昼/夜、廊下、屋上）
- 屋外（公園、街中、海辺）
- ファンタジー（森、城、ダンジョン）

### 4.2 背景キャプチャ（既存画像から背景を作る）

写真や既存イラストを元にアニメ風の背景を生成できます。

```
【図を挿入: 背景キャプチャ設定画面】
```

1. 「参考画像から背景を生成」をオンに
2. 参考画像を指定
3. 「人物を自動除去」をオンに（推奨）
4. 変形指示を入力（オプション）

**変形プリセット:**
| プリセット | 効果 |
|------------|------|
| アニメ調 | クリーンな線、鮮やかな色に変換 |
| 水彩画風 | 柔らかいタッチに変換 |
| 夕暮れ | オレンジ〜ピンクのトーンに変更 |
| 夜景 | 夜のシーンに変更 |
| 冬景色 | 雪を追加、青系のトーンに |
| サイバーパンク | ネオンライト、未来感を追加 |

### 4.3 インフォグラフィックを作る

キャラクターを中心に据えた図解を作成します。設定資料や紹介画像に最適です。

```
【図を挿入: インフォグラフィック設定画面】
```

**操作手順:**
1. 「インフォグラフィック」ボタンをクリック
2. タイトルを入力
3. メインキャラ画像を指定
4. セクション（情報項目）を追加：
   - タイトル（例：「特技」「趣味」「経歴」）
   - 説明文
   - 配置位置（1〜8 or おまかせ）

**スタイル選択:**
| スタイル | 特徴 |
|----------|------|
| グラレコ風 | カラフルで楽しい雰囲気 |
| ノート風 | 手書きノートのような整理感 |
| マインドマップ風 | 中心から放射状に広がる |
| ホワイトボード風 | 会議で使うホワイトボードのような見た目 |

## 5. シーンを演出する

キャラクターと背景を組み合わせて、1枚の完成画像を作ります。

### 5.1 バトルシーン

2キャラクターの対戦シーンを作成します。

```
【図を挿入: バトルシーン設定画面】
```

**操作手順:**
1. 「シーン構築」ボタンをクリック
2. 「バトルシーン」タブを選択
3. 背景を設定（画像指定 or 説明で生成）
4. 左右それぞれのキャラクターを設定
5. 演出効果を選択

**演出設定:**
| 設定 | オプション |
|------|-----------|
| 衝突演出 | 中央衝突、ビーム衝突、格闘など |
| 優勢設定 | 互角、左優勢、右優勢 |
| 境界効果 | 火花と稲妻、オーラ衝突など |

### 5.2 ストーリーシーン

日常会話や静かな場面を作成します（最大3人まで）。

```
【図を挿入: ストーリーシーン設定画面】
```

**キャラクター設定:**
- 画像パス
- 表情（笑顔、困惑、真剣など）
- 説明（任意）

**レイアウト設定:**
| 設定 | オプション |
|------|-----------|
| レイアウト | 並んで歩く、向かい合う、背中合わせなど |
| 距離感 | 親しい、普通、離れている |

**演出:**
- 各キャラクターのセリフ
- シーン全体のナレーション

### 5.3 文字で演出する（装飾テキスト）

ゲームUI風の装飾テキストを生成できます。バトルシーンの技名表示やセリフ演出に最適です。

```
【図を挿入: 装飾テキスト設定画面】
```

**テキストタイプ:**

| タイプ | 用途 | カスタマイズ項目 |
|--------|------|------------------|
| **技名テロップ** | 必殺技名の表示 | フォント、サイズ、グラデーション、縁取り、発光効果 |
| **決め台詞** | インパクトのあるセリフ | 書き文字風、縦書き叫び、ポップ体、回転・変形 |
| **キャラ名プレート** | キャラクター紹介 | フォント、装飾スタイル |
| **メッセージウィンドウ** | ゲーム風セリフ枠 | 名前+顔アイコン+セリフのフル仕様から選択可 |

**技名テロップの演出例:**
- フォント: 極太明朝、筆文字、ゴシック
- グラデーション: 白→青、金→オレンジ、虹色
- 発光効果: 青い稲妻、炎、電撃

## 6. 素材を仕上げる・組み立てる

生成した画像を加工・編集して、作品を完成させます。

### 6.1 漫画ページコンポーザー（コマ割り・吹き出し）

生成した画像を組み合わせて漫画ページを作成します。

```
【図を挿入: 漫画ページコンポーザー画面】
```

**使い方:**
1. 「漫画コンポーザー」ボタンをクリック
2. テンプレートを選択：
   - 4コマ（16:9縦並び）
   - 8コマ（4:3 2列）
3. 各コマに画像を配置
4. 吹き出しを追加（任意）：
   - 丸（通常）
   - 角丸四角
   - ギザギザ（叫び）
   - 思考（雲）
5. セリフを入力
6. 「ページ生成」で1枚の画像として出力

**ポイント:**
- 漫画コンポーザーは**手動で画像を配置**するツールです
- AIが直接4コマを描く「4コマ漫画生成」機能とは別物です（後述）

### 6.2 背景透過（立ち絵素材化）

キャラクター画像の背景を透過して、合成用の素材にします。

```
【図を挿入: 背景透過ツール画面】
```

**使い方:**
1. 「背景透過」ボタンをクリック
2. 入力画像を指定
3. 除去する背景色を選択：
   - 白（最も一般的）
   - 黒
   - 緑（グリーンバック）
   - 青（ブルーバック）
4. 許容値を調整（端のにじみ具合）
5. 実行

**用途:**
- シーン合成用の立ち絵素材作成
- 漫画コンポーザーへの配置用
- 外部ツールでの合成用

### 6.3 スタイル変換

既存のキャラクターを別のスタイルに変換します。

```
【図を挿入: スタイル変換設定画面】
```

**ちびキャラ化:**
- 2〜3頭身のかわいいデフォルメキャラに
- グッズ用、マスコット用に最適
- 衣装・ポーズの保持オプションあり

**ドットキャラ化:**
- レトロゲーム風のピクセルアートに
- スタイル: 8bit風 / 16bit風 / モダンピクセル
- スプライトサイズ: 32x32 / 64x64 / 128x128

### 6.4 画像加工（修正指示）

「おしい！」画像を「完璧」に近づけます。

```
【図を挿入: 画像加工ダイアログ】
```

**使い方:**
1. 修正したい画像をプレビューに表示
2. 「画像を加工」ボタンをクリック
3. 加工モードを選択：
   - **全体改善**: 画像全体を再生成
   - **部分修正**: 指定した部分だけ変更
4. 指示を入力（例：「目を開けて」「背景をぼかす」）
5. 実行

**よくある修正例:**
- 「表情を笑顔に」
- 「髪の色をもう少し明るく」
- 「背景の人物を消して」
- 「指の本数を修正」

### 6.5 4コマ漫画のAI生成

AIに直接4コマ漫画を描かせる機能です。

```
【図を挿入: 4コマ漫画設定画面】
```

**漫画コンポーザーとの違い:**

| 機能 | 4コマ漫画生成 | 漫画コンポーザー |
|------|--------------|------------------|
| 方式 | AIが4コマ全体を1枚として生成 | 既存画像を手動で配置 |
| 柔軟性 | AIまかせ | コマごとに自由に画像を選べる |
| 用途 | ネタ出し、ラフ作成 | 完成度の高い作品作り |

**4コマ漫画生成の使い方:**
1. 「4コマ漫画」ボタンをクリック
2. 登場人物を設定（名前、説明、参照画像）
3. 各コマの内容を入力（起承転結）
4. セリフを設定
5. 「YAML生成」→「生成」

## 7. 実践ワークフロー

### 7.1 小説の表紙・挿絵を作る

```
【図を挿入: 表紙作成のワークフロー図】
```

**手順:**
1. **キャラデザ**: 顔→素体→衣装→ポーズの順でメインキャラを作成
2. **背景作成**: クライマックスの場所を背景生成で作る
3. **合成**: シーン構築でキャラと背景を合わせる
4. **文字入れ**: 装飾テキストでタイトルロゴを作成
5. **仕上げ**: 必要に応じてスタイル変換（水彩画風など）

**ポイント:**
- 表紙は**縦長（9:16 or 3:4）**がおすすめ
- 解像度は**2K**で出力
- タイトルロゴ用の余白を上部に確保

### 7.2 漫画を作る

```
【図を挿入: 漫画作成のワークフロー図】
```

**方法A: 手軽に4コマを作る（AI生成）**
1. 「4コマ漫画」機能でAIに直接描かせる
2. ネタ出しやラフ作成に最適
3. セリフや構図の確認用に

**方法B: 本格的に漫画を作る（コンポーザー）**
1. 各コマの画像を個別に生成
2. 背景透過で合成用素材にする
3. 漫画コンポーザーでコマ割りして配置
4. 吹き出しとセリフを追加
5. 1枚の漫画ページとして出力

**使い分け:**
- **アイデア出し段階**: AI生成の4コマ
- **完成品を作る段階**: コンポーザーで手動配置

### 7.3 設定資料集を作る

```
【図を挿入: 設定資料作成のワークフロー図】
```

**手順:**
1. **三面図**: 正面・横顔・全身を生成して保存
2. **表情差分**: 同じ角度で表情違いを複数作成
3. **スタイル変換**: ちびキャラ版を作成
4. **インフォグラフィック**: キャラ紹介ページを作成
5. **世界観**: 背景生成で主要な舞台を作成

**ポイント:**
- 三面図は**3:1アスペクト比**を使用
- **同じ衣装三面図**から角度変更すると一貫性が保たれる
- インフォグラフィックで設定を1枚にまとめる

## 8. トラブルシューティング

### Q. 指の本数や関節がおかしいです

**A.** これはAI画像生成の技術的限界です。以下の対処法をお試しください：

- 「画像加工」で手の部分だけ再生成
- 構図を変えて手を見えなくする
- 手を体の後ろに隠すポーズを指定

### Q. さっき作ったキャラと顔が違って見えます

**A.** AIは完全な一貫性を保証できません。以下の工夫が有効です：

- **識別しやすい特徴**を必ず入れる（髪飾り、ホクロ、特定の服の色）
- **三面図ワークフロー**を守る（顔→素体→衣装の順）
- **参照画像**として前のステップの出力を必ず指定する

### Q. 指定した色が別の場所にも広がってしまいます

**A.** 「色移り」現象です。以下で軽減できます：

- 「青いドレスと白い帽子」のように、**部位ごとに色を明記**
- 複雑な色指定は分けて生成し、後から合成

### Q. 「生成できませんでした」と表示されます

**A.** Gemini APIのSafety Filterが作動した可能性があります：

- プロンプトの表現を変える（過激な表現を避ける）
- 衣装の露出度を下げる
- 暴力的な描写を控えめにする

### Q. 生成に時間がかかりすぎます

**A.** 通常10〜60秒程度かかります。以下を確認してください：

- インターネット接続は安定していますか？
- 解像度を1Kに下げてみる
- 複雑なシーン合成は時間がかかります

### Q. 日本語のニュアンスが通じません

**A.** AIは英語ベースで学習されています。以下を試してください：

- より具体的・直接的な表現に言い換える
- 単語を区切って箇条書きにする
- 「〜な感じ」より「〜である」と断定的に書く

### Q. 背景透過がうまくいきません

**A.** 以下を確認してください：

- 背景色が選択した色（白/黒/緑/青）と一致していますか？
- 許容値を上げると、多少色が違っても透過されます
- グラデーション背景には対応していません

### 9.2 一貫性を保つコツ

1. **特徴的なアクセサリー**を必ず入れる
2. **同じ描画スタイル**を選び続ける
3. **三面図ワークフロー**を飛ばさない
4. **参照画像**を必ず指定する

### 9.3 生成回数を節約するコツ

- まず**1K解像度**でイメージを固める
- 満足したら**清書モード**で2K化
- **YAML保存**で成功パターンを再利用
